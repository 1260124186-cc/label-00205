1、数据预处理。对数据进行标准化归一化处理，然后通过孤立森林算法进行异常值检测，找出异常数据并移动到异常数据表中；对缺失数据进行插值处理，基于时间序列的时间间隔进行插值；使用卡尔曼滤波对数据进行数据平滑处理。 
2、根据螺栓的历史数据变化趋势，预测单个螺栓的状态，其状态包括正常、预警、故障。状态包括： 
(1) 正常情况，就是根据历史的数据判定螺栓预紧力处于一个稳定区间内；轻微偏离，即预紧力开始出现波动但仍在安全范围内；明显偏离，预紧力显著变化需引起关注。通过LSTM神经网络对时间序列数据建模，结合温度与预紧力的耦合关系，构建多变量预测模型，实现对未来一段时间螺栓状态的精准预测，并将结果写入状态表供产线系统调用，为智能制造提供实时决策支持。 
(2) 关注级预警，数据变化趋势出现有一些与前一段时间的不同情况，需要关注，比如看看后续几天的情况。是因为预紧力的某个变化趋势通过算法得出的结论，对应的处理方法：加强监测频率，记录异常特征。 
(3) 检查级预警，数据的异常变化趋势延续了一段时间，比如3天，需要现场检查。以判定异常类型，并确定解决方案。对应的处理方法是：组织专业检查并制定维护方案。 
(4) 紧急级预警，未来可能发生故障的情况，包括松动、过载、断裂。对应的处理方法是：立即实施处理措施，防止事故扩大。 
① 松动表示预紧力显著下降且低于某个阈值。这个阈值可以人工指定，也可以模型动态设置。也可以进行动态判断而不用阈值； 
② 过载表示预紧力显著上升并超过材料屈服极限或设计允许最大值，可能引发连接件塑性变形或疲劳损伤； 
③ 断裂则表现为预紧力骤降至零。 
(5) 故障，已经发生故障的情况，包括松动、过载、断裂。必须立即进行处理。 
以上5种状态，包括1个正常、3种类型预警、1种故障，数据模型还要分别针对3种预警和1种故障给出诊断结论和推荐措施。可以结合大模型的接口来整理和总结。

13、预测当前状态。用户可以在配置文件中配置自动执行任务的时间和频率，以从mysql数据表中获取数据并加载到模型进行预测，并将预测结果保存到数据库表中。查询螺栓数据的SQL语句如下： 

SELECT   
id,    create_time,    sensor_id,    ptf 
FROM  ( 
SELECT   
id,        create_time,        sensor_id,        ptf, 
@ rank  :=  IF (@current_sensor = sensor_id, @ rank  + 1, 1)  AS  sensor_rank, 
@current_sensor := sensor_id 
FROM  sc_bolt_data 
CROSS   JOIN  ( SELECT  @current_sensor :=  NULL , @ rank  := 0)  AS  vars 
ORDER   BY  sensor_id, create_time  DESC 
)  AS  ranked_data 
WHERE  sensor_rank <= 100 
ORDER   BY  sensor_id, create_time  DESC ; 

预紧力表sc_bolt_data的部分字段如下： 
1）id  主键，自动增长  bigint 
2）sensor_id  ,通道id， 螺栓id   bigint 
3）collector_id  ， 采集器id，bigint 
4）splitter_num  ， 分线器id，bigint 
5）position  ，安装位置   varchar(200) 
6）ptf  ， 预紧力   double 
7）create_time  ,创建时间  datetime 
上表中，collector_id、splitter_num、position三个字段共同拼接组成一个法兰面（字符串拼接逻辑是：采集器id + '-' + splitter_num + ‘-’+安装位置，比如采集id=123，分线器id=456， 安装位置=’B面’，则法兰面=’123-456-B面’），即一个安装位置。 

查询结果整理后作为模型的输入参数给到模型进行预测。 
预测结果写入以下表中： sci_abnormal_prediction: 
1) id 序号主键，自增长bigint 
2) bolt_id 螺栓编码 bigint  
3)  flm_id 法兰面组编码 varchar(100) 
4)  node_type 节点类型 螺栓、法兰面 varchar(6) 
5)  year_month 年月 char(6) 
6)  pw_type 预警预测类型 正常、关注级预警、检查级预警、紧急级预警、故障 varchar(10) 
7)  begin_time 预计发生起始时间 datetime 
8)  end_time 预计发生结束时间 datetime 
9)  confidence 预测置信度 比如：90%，为0.9 float 
10)  rec_measures 推荐措施 文字描述 varchar(1000) 
11)  recent_time 状态时间 螺栓取数的最近的时间,基于这个时间往前找N(默认值100)条原始记录。 datetime 
12) create_time 创建时间 datetime 

法兰面的预测，则依赖这个法兰面上每个螺栓的最近的数据。所以从sc_bolt_data表中查询一个法兰面的每个螺栓的最近200条状态记录。参考SQL语句如下： 
SELECT   
id, 
sensor_id, 
collector_id, 
splitter_num, 
position , 
CONCAT (collector_id, '-', splitter_num, '-',  POSITION )  AS  flang, 
ptf, 
create_time 
FROM  ( 
SELECT   
*, 
@rn :=  IF (@sensor_id = sensor_id, @rn + 1, 1)  as  rn, 
@sensor_id := sensor_id 
FROM  ( 
SELECT   
id, 
sensor_id, 
collector_id, 
splitter_num, 
position , 
ptf, 
create_time 
FROM  sc_bolt_data 
WHERE   CONCAT (collector_id, '-', splitter_num, '-',  position ) = '123-456-B面'  --替换为具体的法兰面 
ORDER   BY  sensor_id, create_time  DESC 
) sorted, 
( SELECT  @rn := 0, @sensor_id :=  NULL ) vars 
) ranked 
WHERE  rn <= 200 
ORDER   BY  sensor_id, create_time  DESC ; 

查询结果整理后作为模型的输入参数给到模型进行预测。

## 4. 提示词内容

基于机器学习的螺栓预紧力预测系统，实现：

1. **数据预处理模块**
   - 数据标准化处理
   - 孤立森林算法异常值检测
   - 时间序列插值补齐
   - 卡尔曼滤波数据平滑

2. **螺栓状态预测模型**
   - LSTM神经网络：Input(100×2) → LSTM(128) → Dropout(0.2) → LSTM(64) → Dropout(0.2) → FC(32) → Output(5类)
   - 状态类别：正常、关注级预警、检查级预警、紧急级预警、故障
   - 考虑温度耦合影响

3. **风险评估模块**
   - 贝叶斯网络风险评分（1-10分）
   - 风险等级：低(8-10)、中(4-7)、高(1-3)
   - 生成诊断结论和改善建议

4. **法兰面状态预测**
   - 注意力机制的多螺栓关联分析
   - 螺栓特征提取 → 自注意力 → LSTM → 输出(5类)

5. **月度趋势预测**
   - Prophet时间序列预测未来30天
   - 识别松动、过载、断裂等异常类型

6. **API服务**
   - 螺栓预测接口（接收最近100条数据）
   - 法兰面预测接口（接收最近200条数据）
   - 定时任务自动训练和预测
