# 螺栓预紧力机器学习预测系统 - 需求规格说明书

## 1. 项目概述

### 1.1 项目目标
开发一个基于机器学习的螺栓预紧力预测系统，能够：
- 基于历史数据训练预测模型
- 预测单个螺栓的状态（正常、预警、故障）
- 预测法兰面组的整体状态
- 提供风险评估和改善建议
- 以API服务方式提供预测能力

### 1.2 数据源
- `data/data_bolt.csv`: 单螺栓预紧力数据
- `data/data_flm.csv`: 法兰面多螺栓预紧力数据
- MySQL数据库: `bolt_preload` (127.0.0.1, root, 无密码)

## 2. 功能需求

### 2.1 数据预处理模块
- **标准化归一化**: 对预紧力数据进行标准化处理
- **异常值检测**: 使用孤立森林算法检测异常数据
- **缺失值插值**: 基于时间序列进行插值处理
- **数据平滑**: 使用卡尔曼滤波进行数据平滑

### 2.2 螺栓状态预测
状态类别：
1. **正常**: 预紧力处于稳定区间
2. **关注级预警**: 数据变化趋势异常，需要关注
3. **检查级预警**: 异常趋势持续3天以上，需现场检查
4. **紧急级预警**: 可能发生松动、过载、断裂
5. **故障**: 已发生松动、过载或断裂

### 2.3 法兰面状态预测
- 基于多螺栓数据的关联分析
- 使用注意力机制捕捉螺栓间依赖关系
- 状态类别与螺栓相同

### 2.4 风险评估
- 风险评分: 1-10分（分数越低风险越高）
- 风险等级: 低(8-10)、中(4-7)、高(1-3)
- 结合贝叶斯网络与LSTM模型输出

### 2.5 预测策略配置
- **策略一**: 应报尽报，允许低误报率
- **策略二**: 精准报警，允许低漏报率

## 3. 技术架构

### 3.1 螺栓模型架构
```
输入层(100×2) → LSTM(128) → Dropout(0.2) 
→ LSTM(64) → Dropout(0.2) → FC(32) → 输出层(5类)
```

### 3.2 法兰面模型架构
```
螺栓特征提取 → 自注意力机制 → LSTM层 
→ 全连接层 → 输出层(5类)
```

### 3.3 特征工程
- 时序特征: 滑动窗口统计量、趋势特征、周期性特征
- 统计特征: 分布特征、变化率特征、极值特征
- 领域特征: 预紧力安全区间、历史故障模式

### 4.1 螺栓预测接口
```json
{
  "螺栓id": "B001",
  "data": [
    ["20250201 00:00:00", 400.00],
    ["20250201 00:01:00", 400.00]
  ]
}
```

### 4.2 法兰面预测接口
```json
{
  "法兰面id": "F001",
  "data": [

    [["20250201 00:00:00", 598], ["20250201 00:01:00", 594]]
  ]
}
```

## 5. 数据库表结构

### 5.1 预测结果表 (sci_abnormal_prediction)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| bolt_id | bigint | 螺栓编码 |
| flm_id | varchar(100) | 法兰面编码 |
| node_type | varchar(6) | 节点类型 |
| year_month | char(6) | 年月 |
| pw_type | varchar(10) | 预警类型 |
| begin_time | datetime | 预计起始时间 |
| end_time | datetime | 预计结束时间 |
| confidence | float | 置信度 |
| rec_measures | varchar(1000) | 推荐措施 |
| recent_time | datetime | 状态时间 |
| create_time | datetime | 创建时间 |

### 5.2 月度预测表 (ci_month_prediction_details)
结构与 sci_abnormal_prediction 相同
