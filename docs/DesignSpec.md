## 2. 模块设计

### 2.1 数据预处理模块 (preprocessing)
- `StandardScaler`: 标准化处理
- `IsolationForestDetector`: 孤立森林异常检测
- `TimeSeriesInterpolator`: 时间序列插值
- `KalmanSmoother`: 卡尔曼滤波平滑

### 2.2 特征工程模块 (feature_engineering)
- `TimeSeriesFeatures`: 时序特征提取
- `StatisticalFeatures`: 统计特征提取
- `DomainFeatures`: 领域特征提取

### 2.3 模型模块 (models)
- `BoltLSTMModel`: 螺栓LSTM预测模型
- `FlangeAttentionModel`: 法兰面注意力模型
- `BayesianRiskModel`: 贝叶斯风险评估模型
- `ProphetForecast`: Prophet时间序列预测

### 2.4 服务模块 (services)
- `PredictionService`: 预测服务
- `TrainingService`: 训练服务
- `RiskAssessmentService`: 风险评估服务

### 2.5 API模块 (api)
- `BoltPredictionAPI`: 螺栓预测接口
- `FlangePredictionAPI`: 法兰面预测接口
- `RiskAssessmentAPI`: 风险评估接口

### 2.6 调度模块 (schedulers)
- `TrainingScheduler`: 模型训练调度
- `PredictionScheduler`: 预测任务调度

## 3. 状态定义

### 3.1 状态枚举
| 状态码 | 状态名称 | 说明 |
|--------|----------|------|
| 0 | 正常 | 预紧力处于稳定区间 |
| 1 | 关注级预警 | 趋势异常，需关注 |
| 2 | 检查级预警 | 持续异常，需检查 |
| 3 | 紧急级预警 | 可能故障，需紧急处理 |
| 4 | 故障 | 已发生故障 |

### 3.2 故障类型
| 类型码 | 类型名称 | 特征 |
|--------|----------|------|
| L | 松动 | 预紧力显著下降 |
| O | 过载 | 预紧力超过设计值 |
| B | 断裂 | 预紧力骤降至零 |

## 4. 预警策略配置

### 策略一: 应报尽报
- 误报率阈值: < 5%
- 漏报率: 无限制
- 适用场景: 高安全要求环境

### 策略二: 精准报警
- 误报率: 0%
- 漏报率阈值: < 10%
- 适用场景: 减少无效巡检

## 5. 技术栈

- **框架**: FastAPI, PyTorch, scikit-learn
- **数据库**: MySQL, SQLAlchemy
- **调度**: APScheduler
- **时间序列**: Prophet
- **容器化**: Docker, docker-compose
